Concept Explanation

For 4-year-olds, abstract numbers are difficult to grasp. This activity uses One-to-One Correspondence, where the child physically touches one object as they say one number, grounding mathematical concepts in physical reality.

Suggested Classroom Activity

  • 01 "Planting Seeds": Give the child flower pots labeled 1-5. Ask them to "plant" the correct number of wooden beads or beans in each.
  • 02 "Sunlight Counting": Use yellow cut-out suns and have them count out 'rays' (clothespins) to clip onto the sun based on the number written in the center.
